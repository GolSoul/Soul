# Soul - Oracle Precision Pro Trading Strategy

A sophisticated Pine Script v5 trading strategy for TradingView featuring advanced confidence scoring, multi-timeframe analysis, and comprehensive risk management.

## 🎯 Strategy Features

- **Customizable Confidence Scoring**: Weight-based system combining trend, MACD, RSI, ATR, and ADX
- **Multi-Timeframe Analysis**: Higher timeframe filter for trend alignment
- **Advanced Risk Management**: ATR-based and percentage-based stops with position sizing
- **Enhanced Visuals**: Clear entry/exit signals and trend background coloring
- **Detailed Alerts**: Rich notifications with price levels and confidence scores
- **Backtesting Ready**: Full Strategy Tester compatibility
